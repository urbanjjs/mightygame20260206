# 마이티 온라인 게임 - 테스트 버전

한국의 전통 카드 게임 마이티를 웹 브라우저에서 즐길 수 있는 테스트 버전입니다.

## 🎮 실행 방법

1. 모든 파일을 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. 게임이 자동으로 시작됩니다.

## 📋 게임 규칙

### 기본 규칙
- 5인 플레이 (1명 사용자 + 4명 AI)
- 53장의 카드 사용 (트럼프 52장 + 조커 1장)
- 각 플레이어에게 10장씩 배분, 바닥에 3장
- 총 10라운드 진행

### 팀 구성
- **여당**: 주공(1명) + 친구(1명)
- **야당**: 나머지 3명
- 친구는 지정된 카드를 낼 때까지 비공개

### 점수 카드
- A, K, Q, J, 10이 각 1점 (총 20점)

### 특수 카드
- **마이티**: 게임 내 최강 카드 (기본 ♠A, 기루다가 ♠면 ◆A)
- **조커**: 마이티 다음으로 강력 (1, 10라운드 제외)
- **조커콜**: 조커를 강제 출전시킴 (기본 ♣3)

### 비딩
- 13부터 20까지 비딩 가능
- 숫자가 같으면 수트 강도로 비교 (♠ > ◆ > ♥ > ♣ > No)

### 점수 계산 (10점 빼기 방식)
- 여당 승리: (여당 득점 - 10)점씩 획득
- 야당 승리: (공약 - 여당 득점)점씩 지불
- 백런(야당 20점 전부 획득): 여당 특별 패널티
- 노기루다: 점수 2배

## 🔧 테스트 모드

테스트 모드를 활성화하면:
- ✅ AI의 손패가 공개됩니다.
- ✅ AI의 의사결정 로그가 표시됩니다.
- ✅ 낼 수 있는 카드가 하이라이트됩니다.

우측 상단의 "🔧 테스트모드" 버튼으로 ON/OFF 가능합니다.

## ⚙️ 설정

⚙️ 설정 버튼을 클릭하여:
- AI 카드 공개 여부
- AI 의사결정 로그 표시
- 낼 수 있는 카드 표시
- AI 턴 속도 (즉시 / 빠름 / 보통 / 느림)

를 조절할 수 있습니다.

## 🎯 주요 기능

### ✅ 완성된 기능
1. **카드 배분 및 딜미 체크**
   - 자동 카드 배분
   - 딜미(Deal Me) 선언 가능
   - 딜미 풀 누적

2. **비딩 시스템**
   - 13~20까지 비딩
   - 수트 강도 비교
   - 노기루다(No) 비딩

3. **카드 교환**
   - 주공 바닥 카드 획득
   - 3장 선택 교환

4. **친구 선언**
   - 원하는 카드 지정
   - 노프렌드 선언

5. **게임 플레이**
   - 10라운드 진행
   - 특수 카드 (마이티, 조커, 조커콜)
   - 조커 수트 선언
   - 트릭 승자 결정

6. **AI 시스템**
   - 비딩 전략
   - 친구 선택 전략
   - 카드 교환 전략
   - 플레이 전략 (여당/야당 구분)
   - 의사결정 로그

7. **점수 계산**
   - 10점 빼기 방식
   - 백런 체크
   - 노기루다 2배
   - 노프렌드 점수 조정

8. **테스트 기능**
   - AI 카드 공개
   - 의사결정 로그
   - 낼 수 있는 카드 하이라이트

## 🎲 게임 진행 순서

1. **카드 배분** → 각 플레이어 10장, 바닥 3장
2. **딜미 체크** → 너무 약한 패면 다시 배분 가능
3. **비딩** → 공약 선언 (13~20)
4. **카드 교환** → 주공이 바닥 3장 받고 3장 버림
5. **친구 선언** → 주공이 친구 카드 지정
6. **게임 플레이** → 10라운드 진행
7. **결과 계산** → 점수 계산 및 표시

## 🤖 AI 전략

### 비딩 전략
- 손패 평가 점수 계산
- 마이티, 조커 보유 시 가산점
- 긴 수트 우선 기루다 선택

### 플레이 전략 (여당)
- 공약 달성 여부에 따른 공격/수비
- 마이티 후반부 활용
- 기루다 높은 카드 우선 리드

### 플레이 전략 (야당)
- 주공 약점 공략
- 점수 카드 최소화
- 팀원 트릭 승리 시 점수 추가
